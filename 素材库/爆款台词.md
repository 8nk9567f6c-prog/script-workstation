## 一、大女主觉醒类

### 核心情绪：从"被选择"到"我选择"

| 台词 | 出处 | 适用场景 | 复用模板 |
|------|------|----------|----------|
| "请务必一而再再而三地救自己于水火。" | 《好一个乖乖女》 | 女主觉醒转折点 | "请务必___"——万能觉醒句式 |
| "她不是西子城的西子，她是妖都的野王，旷野鹿王！" | 《好一个乖乖女》 | 配角揭示女主真实身份 | "[名字]不是[表面身份]，ta是[真实身份]！" |
| "不管是分手还是决裂，麻烦你，风光又体面地活着，活给我看！" | 《好一个乖乖女》 | 诀别/分手场景 | "麻烦你，___地活着，活给我看" |
| "为什么女人不管在外面混得多好，回家还是要洗碗端菜？" | 《好一个乖乖女》 | 社会议题共鸣场景 | 用具体生活细节质问不公——不需要回答，问题本身就是力量 |

### 大女主台词设计公式
```
本集女主金句 = 一个被轻视的处境 + 一个出人意料的回应 + 一个让观众想截图的画面
例：前夫甩离婚协议 → 她签完字笑了 → "谢谢，我等这一天等了三年。"
```

## 二、忠犬/霸总护妻类

### 核心情绪：上位者为爱低头

| 台词 | 出处 | 适用场景 | 复用模板 |
|------|------|----------|----------|
| "我是你的狗！" | 《栀栀复栀栀》第5集 | 男方放下身段的极致表白 | "我是你的___"——卑微到尘埃的告白句式 |
| "西门礼臣，只为你活着。" | 《栀栀复栀栀》第7集 | 校园/纯爱告白 | "[名字]，只为___"——专属承诺句式 |
| "家规，我西门礼臣自成一家，家规只有一条——惹江晚栀，不行！" | 《栀栀复栀栀》第42集 | 当众对抗家族/权威护妻 | "规矩只有一条——惹___，不行！" |
| "搞钱不如搞我。" | 《栀栀复栀栀》 | 霸总撩人 | "___不如___"——自信式告白 |
| "这样，我的每一张钞票上都写你的名字。" | 《栀栀复栀栀》 | 求婚/告白 | 用具体物质+抽象承诺的组合——"我的每一___都写你的名字" |
| "栀栀，再骗骗我，你再骗骗我。我很好骗的。" | 《栀栀复栀栀》第65集 | 脆弱与偏执的极致 | "你再骗骗我，我很好骗的"——爱到甘愿被骗的绝望告白 |
| "我爱你，快要疯了。" | 《好一个乖乖女》 | 男主情感爆发点 | "我爱ta，快要___" |
| "再利用我一次！踩着我的肩膀竭尽所能地爬到你所能爬到的最高点！" | 《好一个乖乖女》 | 托举式爱情——男主甘愿做垫脚石 | "踩着我的___，爬到___" |

### 霸总台词设计公式
```
本集霸总金句 = 一个绝对的身份优势 + 一个为爱放弃优势的动作 + 一个让观众嗑到的反差
例：他是京城最有权势的男人 → 他跪下来给她系鞋带 → "你踩过的地，我都觉得是干净的。"
```

## 三、复仇打脸类

### 核心情绪：忍辱后的爆发、迟到的正义

| 台词 | 出处 | 适用场景 | 复用模板 |
|------|------|----------|----------|
| "三年前你杀我的时候，想过今天吗。" | 《重生千亿》 | 复仇高潮 | "[时间]前你[做过的事]，想过今天吗" |
| "这婚，我退定了。" | 短剧通用句式 | 女主决裂宣言 | "这___，我___定了"——万能决裂句式 |
| "你以为的金手指，其实是别人的玩具。" | 行业通用金句 | 反转揭示——主角的能力原来是别人给的 | "你以为的___，其实是别人的___" |

### 复仇台词设计公式
```
本集复仇金句 = 回顾历史伤害（让观众想起为什么恨反派）+ 现在的位置对调 + 一个留白让观众自己补完
例："你还记得你把我扔在雨里的那天吗？" → 她坐在加长劳斯莱斯里，车窗缓缓升上 → "今天，轮到你在雨里走了。"
```


## 六、海外短剧出圈台词（中式霸总出海）

| 台词（英文） | 中式理解 | 短剧化用 |
|-------------|---------|----------|
| "You think you're the first girl to try and get my attention?" | "女人，你在勾引我。" | 霸道总裁初见台词 |
| "I told you not to betray me!" | "我告诉过你，不许背叛我！" | 发现背叛时的台词 |

**启示：** 中式霸总台词的"晋江味"在海外有巨大市场——写台词时不用过度本地化，保持"中式暧昧的含蓄+霸总的直接"这种矛盾感本身就是卖点。

## 七、台词创作自检清单

写完每集台词后对照检查：

```
[ ] 本集有没有一句能让观众想截图传播的金句？
[ ] 台词是否口语化（单句≤15字，不用书面语）？
[ ] 每句台词是否在推进情节或揭示角色？（废话台词=0）
[ ] 有没有"AI腔"连接词（岂料/殊不知/与此同时）？
[ ] 情感爆发点是否有"一句封神"的潜质？
[ ] 角色台词是否符合其人设和身份？（霸总不能说怂话，女主觉醒前不能太犀利）
[ ] 有没有"情绪金句"——让观众在评论区反复引用的那句话？
```

## 八、金句储备公式（创作时直接套用）

### 万能反转句
```
"我以为___，原来___。"
例："我以为你是来救我的，原来你是来杀我的。"
```

### 万能觉醒句
```
"从今天起，我不再做___的___。"
例："从今天起，我不再做你的妻子。"
```

### 万能碾压句
```
"你以为___？不，___。"
例："你以为这是我求你的机会？不，这是你最后的机会。"
```

### 万能情感句
```
"___年了，你终于___了。可惜___。"
例："三年了，你终于肯看我了。可惜我已经不需要了。"
```
