# 编剧工作台 v4.0 — 网页端使用说明

## 访问方式

### 电脑端
直接双击打开：`C:\Users\17928\Desktop\编剧工作台\web\index.html`

或在浏览器地址栏输入：`file:///C:/Users/17928/Desktop/编剧工作台/web/index.html`

### 手机端（两种方式）

**方式一：本地网络访问（推荐，同一WiFi下）**
1. 在电脑上打开命令行，进入 web 目录：
   `cd C:\Users\17928\Desktop\编剧工作台\web`
2. 启动本地服务器（Python）：
   `python -m http.server 8080`
3. 查看电脑 IP 地址：命令行输入 `ipconfig`，找到 IPv4 地址
4. 手机连接同一 WiFi，浏览器打开：`http://电脑IP:8080`
5. 在手机浏览器菜单中选择「添加到主屏幕」

**方式二：部署到静态服务器（随时随地访问）**
免费部署平台（任选其一）：
- GitHub Pages：把 web 文件夹上传到 GitHub 仓库，开启 Pages
- Vercel：导入 GitHub 仓库，自动部署，获得 `xxx.vercel.app` 域名
- 腾讯云静态网站托管：上传 web 文件夹即可

## 数据联动机制

网页端通过读取 `data.json` 获取所有规则和工作流数据。

### 更新网页端数据

当你修改了编剧工作台的底层规则（CLAUDE.md），按以下步骤同步网页：

1. 编辑 `C:\Users\17928\Desktop\编剧工作台\web\data.json`
2. 找到对应的规则分类（如 redlines/iron/emotion 等）
3. 修改或新增规则条目（每项含 n/title/desc 三个字段）
4. 更新 `"updated"` 日期
5. 保存文件

网页端有自动缓存机制：
- 打开网页时自动加载最新 data.json
- 点击工作台页面的「从工作台同步最新数据」按钮可手动刷新
- 离线时自动使用浏览器缓存的数据

### 文件结构

```
web/
├── index.html      ← 网页应用主文件
├── data.json       ← 规则数据库（修改此文件即可更新网页内容）
└── manifest.json   ← PWA 配置
```

### 数据联动关系

```
CLAUDE.md  ←── 编剧工作台核心配置（Claude Code 读取）
    │
    └── 手动同步 ──→  web/data.json  ←── 网页应用规则数据
                           │
                           └── 自动加载 ──→  index.html 网页端显示
```

当你通过 Claude Code 更新 CLAUDE.md 的规则后，同步编辑 data.json 中对应内容，网页端即可自动获取最新规则。
