# 编剧工作台 v4.1 — 网页端使用说明

## 访问方式

### 📱 手机端（推荐——零配置直接打开）
**使用独立版文件 `index-standalone.html`**，数据已内嵌在文件中，不需要任何服务器，任何设备直接双击或拖入浏览器即可打开。

1. 将 `index-standalone.html` 通过微信/QQ/AirDrop/USB 发送到手机
2. 在手机文件管理器找到该文件 → 选择用浏览器打开
3. （可选）在浏览器菜单选择「添加到主屏幕」作为快捷方式

### 💻 电脑端
直接双击 `index-standalone.html` 即可在浏览器中打开。

## 网页功能

| 标签 | 功能 |
|------|------|
| 工作台 | 项目创建/切换/进度管理 + 核心信条 + 数据同步 |
| 工作流 | 9步创作命令完整指南 + 产出纪律 + Save the Cat 节拍表 |
| 规则库 | 54条核心规则（写作红线/内容铁律/情绪银行/黄金四步/快准新狠/三密度/钩子/节奏/游戏化/人物/格式） |
| 自检 | 42项交互式自检清单（点击勾选、自动保存到浏览器） |

## 数据联动机制

网页端通过读取 `data.json` 获取所有规则和工作流数据。

### 更新网页端数据

当你修改了编剧工作台的底层规则（CLAUDE.md），按以下步骤同步网页：

1. 编辑 `C:\Users\17928\Desktop\编剧工作台\web\data.json`
2. 找到对应的规则分类
3. 修改或新增规则条目
4. 更新 `"updated"` 日期
5. 保存文件

网页端有自动缓存机制：
- 打开网页时自动加载最新 data.json
- 点击工作台页面的「刷新数据」按钮可手动刷新
- 离线时自动使用浏览器 localStorage 缓存的数据

### 文件结构

```
web/
├── index.html      ← 网页应用主文件
├── data.json       ← 规则数据库（修改此文件即可更新网页内容）
├── sw.js           ← Service Worker（离线缓存）
└── manifest.json   ← PWA 配置
```

### 数据联动关系

```
CLAUDE.md  ←── 编剧工作台核心配置（Claude Code 读取）
    │
    └── 手动同步 ──→  web/data.json  ←── 网页应用规则数据
                           │
                           └── 自动加载 ──→  index.html 网页端显示
```
